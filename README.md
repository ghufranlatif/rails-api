Rails API
==================

## Installation

To install this API on your computer, type the following commands:

```bash
$ cd rails-api
$ bundle install
$ rake db:setup
```

## Running The API

To run this API on your computer (localhost:3000/v1/...), type the following commands:
```bash
$ spring rails server
```

## Running Tests
To run tests on the API, type the following commands:
```bash
$ spring rspec
```
